# NixOS Media Box | NixOS 盒装安装媒介

A toy project, inspired by [arch-media-box-typst](https://github.com/Myriad-Dreamin/arch-media-box-typst) and other similar projects.

The graphic design is entirely done via [Typst](https://typst.app/), with the background style placed in [background.typ](background.typ) and the main content in [box.typ](box.typ). The measurement and layout are pretty rough as well as it haven't actually been verified. Feel free to modify it.

External resources used in this project:

- [NixOS logo](https://nixos.org/branding) (svg)
- Vegur (font)
- Noto Sans CJK SC (font)
